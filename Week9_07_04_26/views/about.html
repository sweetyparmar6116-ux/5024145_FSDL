About Page

This is a simple AngularJS Hello World application with routing and interactive features.

Features Included:

  • ng-app: Initializes the AngularJS application
  • ng-controller: Attaches controller logic to HTML elements
  • ng-model: Binds input values to controller properties
  • Expressions: Displays dynamic values with {{ }}
  • ng-click: Handles button click events
  • ng-repeat: Loops through arrays to display lists
  • Filters: Formats data (e.g., uppercase filter)
  • Routing: Single Page Application (SPA) navigation
